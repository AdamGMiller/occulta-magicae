# Occulta Magicae

A browser-based card game set in the world of Ars Magica, where you guide a magus through seasons of study, exploration, and mystery in Mythic Europe (Anno Domini 1220).

## Getting Started

Open `game/index.html` in a browser to play, or serve the `game/` directory with any static file server.

## Project Structure

- **game/** — The web application (HTML/CSS/JS)
- **image-gen/** — Python scripts for generating card art

## Development

Install dev dependencies from the root or from `game/`:

```bash
npm install
```
